# AI Coding 与前沿 AI 工具使用说明

## 1. 使用范围

项目开发过程中使用 AI Coding 工具辅助代码阅读、方案讨论、单元测试补充、错误定位、文档草拟和界面检查。AI 工具不代替团队对需求、架构、代码合并和参赛陈述的责任。

本次 `main-jsc` 更新使用 Codex 协助完成比赛演示页、运行时配置、测试修复建议和文档整理。项目运行时支持讯飞星火、DeepSeek、OpenAI-compatible、Anthropic 和本地 Ollama；比赛推荐配置讯飞星火。

## 2. 人工复核

所有 AI 生成或修改内容均应经过以下流程：

1. 团队成员确认需求与修改范围。
2. 运行单元测试、集成测试、类型检查和生产构建。
3. 对模型地址、协议、引用和课程事实进行人工核对。
4. 检查提交差异，确认不包含密钥、个人数据和无关文件。
5. 由团队成员完成最终提交、答辩和责任确认。

## 3. 数据与密钥

不向代码仓库提交真实 API 密钥、学生身份信息或私有课程材料。比赛演示使用合成学生画像。输入 AI Coding 工具的内容限定为项目代码、公开赛题和合成数据；敏感信息应先脱敏。

## 4. 自主开发声明

TutorBot 的需求取舍、产品设计、课程知识库、多智能体编排、测试验收和比赛演示由参赛团队负责。AI Coding 工具仅作为开发辅助，不能被视为作品作者或评审结论来源。
