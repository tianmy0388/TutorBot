# 第 1 章：什么是人工智能

## 1.1 定义

**人工智能（Artificial Intelligence, AI）** 是研究如何使计算机系统表现出类似于人类智能行为的一门学科。它涵盖学习、推理、感知、决策、自然语言理解等多种能力。

> 经典定义（John McCarthy, 1956）："制造智能机器的科学与工程，特别是使计算机表现出智能行为的程序。"

## 1.2 AI 的层级关系

```
人工智能 (AI)
├── 机器学习 (Machine Learning, ML)
│   └── 深度学习 (Deep Learning, DL)
│       ├── CNN（计算机视觉）
│       ├── RNN / LSTM（序列建模）
│       └── Transformer / LLM（自然语言）
└── 传统 AI（专家系统、搜索、规划等）
```

- **AI** 是最大范畴
- **ML** 是 AI 的子集：系统从数据中学习而非被显式编程
- **DL** 是 ML 的子集：使用多层神经网络自动学习特征表示

## 1.3 AI 的应用领域

| 领域 | 代表应用 |
|---|---|
| 计算机视觉 | 人脸识别、自动驾驶、医学影像 |
| 自然语言处理 | 机器翻译、文本生成、智能客服 |
| 语音 | 语音识别、TTS、智能音箱 |
| 推荐系统 | 电商、视频、信息流 |
| 决策与博弈 | AlphaGo、机器人控制 |
| 科学发现 | AlphaFold、药物设计 |

## 1.4 关键里程碑

- 1956：达特茅斯会议，AI 学科诞生
- 1997：DeepBlue 击败卡斯帕罗夫
- 2012：AlexNet 在 ImageNet 夺冠，深度学习崛起
- 2016：AlphaGo 击败李世石
- 2017：Transformer 架构提出（"Attention Is All You Need"）
- 2020：GPT-3 展现 Few-shot Learning 能力
- 2022：ChatGPT 引爆 LLM 浪潮

## 1.5 本章小结

- AI ≠ ML ≠ DL，注意区分三者层级
- AI 已经从规则系统 → 统计学习 → 深度学习 → 大模型演进
- 接下来从机器学习基础开始深入
