# 第 3 章：神经网络与深度学习

## 3.1 从感知机到神经网络

**感知机（Perceptron）** 是最简单的神经网络单元：

$$y = f\left(\sum_i w_i x_i + b\right)$$

其中 $f$ 是激活函数。

单层感知机只能解决线性可分问题（连 XOR 都学不会），多层感知机（MLP）通过引入隐藏层和激活函数获得非线性建模能力。

## 3.2 多层前馈神经网络

```
输入层 (784) → 隐藏层1 (256) → 隐藏层2 (128) → 输出层 (10)
```

每个隐藏层做仿射变换 + 激活函数：

$$\mathbf{h}^{(l)} = \sigma\left(\mathbf{W}^{(l)} \mathbf{h}^{(l-1)} + \mathbf{b}^{(l)}\right)$$

## 3.3 激活函数

| 名称 | 公式 | 特点 |
|---|---|---|
| Sigmoid | $\sigma(x)=\frac{1}{1+e^{-x}}$ | 输出 (0,1)，易梯度消失 |
| Tanh | $\tanh(x)$ | 输出 (-1,1)，零中心 |
| ReLU | $\max(0, x)$ | 计算快，缓解梯度消失，但"Dying ReLU" |
| Leaky ReLU | $\max(\alpha x, x)$ | 缓解 Dying ReLU |
| GELU | $x \Phi(x)$ | Transformer 中常用 |
| Swish | $x \cdot \sigma(\beta x)$ | 平滑非单调 |

## 3.4 反向传播（Backpropagation）

核心：链式法则 + 计算图。

对损失 $L$，参数 $\theta$ 的梯度：

$$\frac{\partial L}{\partial \theta^{(l)}} = \frac{\partial L}{\partial \mathbf{h}^{(L)}} \cdot \prod_{k=l+1}^{L} \frac{\partial \mathbf{h}^{(k)}}{\partial \mathbf{h}^{(k-1)}} \cdot \frac{\partial \mathbf{h}^{(l)}}{\partial \theta^{(l)}}$$

实现：自动微分（PyTorch `autograd`、TensorFlow `GradientTape`）。

## 3.5 优化器

### 3.5.1 基础：SGD

$$\theta_{t+1} = \theta_t - \eta \nabla L(\theta_t)$$

### 3.5.2 动量（Momentum）

$$v_{t+1} = \beta v_t + \nabla L(\theta_t), \quad \theta_{t+1} = \theta_t - \eta v_{t+1}$$

### 3.5.3 Adam（最常用）

维护一阶矩 $m$（均值）和二阶矩 $v$（方差）的指数移动平均，做偏差校正后更新参数。

### 3.5.4 学习率调度

- Step Decay：每 N 个 epoch 乘 0.1
- Cosine Annealing
- Warmup + Decay

## 3.6 损失函数

- 回归：MSE、MAE、Huber
- 分类：交叉熵（Cross-Entropy）
- 多标签：Binary Cross-Entropy

## 3.7 训练技巧

| 问题 | 解决方案 |
|---|---|
| 梯度消失 | ReLU、残差连接、BatchNorm |
| 梯度爆炸 | 梯度裁剪（Gradient Clipping） |
| 过拟合 | Dropout、权重衰减、数据增强 |
| 训练慢 | Batch Normalization、学习率调度、Adam |
| 鞍点 | SGD 动量、Adam |

## 3.8 PyTorch 实战

```python
import torch
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self, in_dim=784, hidden=256, out_dim=10):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(in_dim, hidden),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(hidden, hidden // 2),
            nn.ReLU(),
            nn.Linear(hidden // 2, out_dim),
        )

    def forward(self, x):
        return self.net(x)

model = MLP()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
```

## 本章小结

- 神经网络的核心是：分层非线性变换 + 反向传播 + 梯度下降
- 激活函数决定了网络的非线性能力
- 优化器和正则化技巧是工程实践的关键
